Para executar o Dashboard:

1 - No terminal ou prompt de comando, instale os pacotes abaixo:

pip install dash
pip install dash_bootstrap_components

2 - No terminal ou prompt de comando, na pasta onde estão os arquivos, execute a data app:

python dataapp.py

3 - Copie o endereço que aparece no terminal e abra no navegador.

4 - Explore a Data App.
